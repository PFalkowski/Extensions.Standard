Lightweight extensions methods (and constants) for common programming tasks, like:

- Suffixes (AsMemory, AsTime)
- Interpolate
- Partition
- Shuffle (O(n))
- Softmax
- InnerProduct
- IsPrime
- ManhattanDistance
- EuclideanDistance
- InRangeInclusive / InRangeExclusive
- MeanSquaredError
- EqualsWithTolerance
- HsVtoArgb
- Scale
- Fit
- DeepCopy (reflection-based deep clone, no dependencies)

and many more.

Extensions for ```System.Random``` (NextBool, NextChar, NextString etc.) moved to [StrongRandom](https://github.com/PFalkowski/StrongRandom)
